CRM Migration & Automation Plan | Pipedrive → Attio → Lead Automation
Project Overview
This document outlines the 90-day plan to migrate the SEO agency’s CRM from Pipedrive to Attio, configure the
system with proper pipeline stages and lead scoring, and implement automated nurture sequences to replace
manual BDR functions.
Key objectives
• Migrate all existing Pipedrive records cleanly to Attio with zero data loss
• Build pipeline stages that map to Danielle’s real-world process
• Implement a 5–7 factor lead scoring model integrating behavioral and firmographic data
• Automate nurture sequences that act as a BDR replacement for warm inbound leads
• Achieve full funnel visibility with reporting on conversion rates by stage
Key milestones
Week 1–2 Audit Pipedrive, export all data, map fields to Attio schema, set up Attio workspace
Week 3–4 Import data, validate records, connect forms & integrations, train Danielle on Attio
Week 5–6 Build pipeline stages, design lead scoring model (5–7 factors), integrate behavioral data
Week 7–8 Integrate firmographic enrichment (PDL / Datagma), build routing rules for SMB vs enterprise
Week 9–10 Build nurture sequences, configure AI-assisted follow-up drafts, set up automation triggers
Week 11–12 QA full funnel, finalise reporting dashboards, hand off to Danielle, 90-day review
